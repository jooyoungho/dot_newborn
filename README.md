# dot_newborn

dot project의 첫 번째 테스트용 코드입니다.

A4 사이즈의 공간에서 랜덤하게 사각형을 출력하며 레이어 합성을 통해 다양한 스타일을 연출할 수 있습니다.

색상 코드는 Adobe Color에서 가져왔습니다.



**사용법**

Xcode → 코드 실행 → 결과를 텍스트 에디터에 붙여넣기 → svg 확장자로 저장
